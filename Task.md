Необходимо создать мобильное приложение на Android, реализующее приложение "Список заметок" со следующим функционалом:
- просмотр списка заметок;
- добавление заметки;
- редактирование заметки;
- удаление заметки;
- удаление нескольких заметок;

При реализации функционала необходимо использовать следующие возможности разработки под Android:
- несколько окон и навигация между ними (например, список заметок - отдельное окно, просмотр заметки - отдельное окно);
- сохранение и загрузка локальных данных (например, использовать для хранения SQLite + Room);
- взаимодействие с правами доступа (например, задействовать при работе с заметками право доступа к файлам, к списку контактов, к геопозиции);
- обработка жестов (например, жест "долгое нажатие" на списке заметок переводит в режим выбора нескольких заметок для их массового удаления).

Дизайн приложения должен поддерживать темную тему и поворот экрана.
Для разработки используется Android Studio (установить также заранее дома Android SDK и AVD).
